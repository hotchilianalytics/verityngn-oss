## Testing Guidelines

### How to Use This List

1. **Baseline Testing**: Start with video #1 (tLJC8hkK-ao) which has already been processed
2. **Category Coverage**: Test at least 2-3 videos from each category
3. **Complexity Levels**: Mix simple, moderate, and complex videos
4. **Verdict Spectrum**: Include videos expected to be True, False, and Mixed

### Expected Outcomes

- **False/Misleading Videos**: Should identify unsubstantiated claims, lack of credible sources, and misleading information
- **True/Legitimate Videos**: Should identify credible sources, scientific backing, and factual claims
- **Mixed Videos**: Should provide nuanced analysis distinguishing between accurate and inaccurate claims

### Validation

Compare VerityNgn results with:
- Existing fact-checks (Snopes, PolitiFact, etc.)
- Scientific consensus (for health/science videos)
- Regulatory warnings (for scam videos)
- Expert analysis (for complex topics)
